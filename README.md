# Enabling Smart Radio Environments in the Frequency Domain With Movable Signals

This repository contains code to reproduce the paper:

> M. Nerini, B. Clerckx, "[Enabling smart radio environments in the frequency domain with movable signals](https://ieeexplore.ieee.org/document/)," IEEE Trans. Wireless Commun., 2026.

## Code

The script `main_Fign.m` reproduces Fig. *n*, for *n* = 4, ..., 16.

The script `main_Fig14.m` needs the files `fig-PR-theta-Ww.mat`, for *w* = 11, 12, 14, 18, which are generated by `main_Fig13.m` with appropriate parameters.

The script `main_Fig16.m` needs the files `fig-PR-thetaR-Ww-thetaTt.mat`, for *w* = 11, 12, 14, 18, and *t* = 0, 1, which are generated by `main_Fig15.m` with appropriate parameters.
